Ella's Notes를 활용하기 위해서는 먼저 정해진 양식대로 작성된 Sheets가 필요합니다.  
아래 가이드를 보고 양식에 맞게 Sheets를 만들어 봅시다.  
- 👉 [Sheets 만들기](google-sheet)

아래 가이드를 참고해서 본인의 Sheets를 사용해 봅시다.
